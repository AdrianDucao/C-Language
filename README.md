# C Language Activities

All syntax and functions you'll ever need for C Programming

## Dependencies

get some compiler

```
Ubuntu
sudo apt install gcc
```

```
Fedora 31
sudo dnf install gcc
```

## Usage
Editing code or use whatever text editor you like.
```
$ nano hellworld.cc
```

Compiling code
```
$ gcc helloworld.c
```

Executing code
```
$ ./helloworld
```
